# Specification: Build core interactive resume and tech stack timeline

## Objective
Create the foundational Next.js application and build the core interactive resume and tech stack timeline features. This directly serves the primary user story: allowing recruiters to quickly evaluate past roles and technical skills.

## Requirements
- Scaffold Next.js application with TypeScript and Tailwind CSS.
- Create a sleek, dark-mode focused layout with header (social links) and footer.
- Build an interactive timeline component for work experience.
- Build a tech stack visualizer showcasing skills.
- Ensure the application is fully responsive.
- Achieve >80% test coverage for the core components.
