# Technology Stack

## Core Language
- **TypeScript:** Chosen for robust static typing, improved developer experience, and scalability.

## Frontend Framework
- **React / Next.js:** Chosen for its industry-standard component architecture, optimized rendering capabilities, and extensive ecosystem.

## Styling
- **Tailwind CSS:** Chosen for utility-first, fast, and consistent styling across the application.

## Deployment & Hosting
- **GitHub Pages:** Chosen for simple, free hosting directly from the repository.
